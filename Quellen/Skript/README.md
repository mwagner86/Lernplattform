# Skript (primäre Wissensbasis / Wahrheitsquelle)

Das **Modulskript** — **kapitelweise als Markdown** (`01-….md`, `02-….md`), **komplett
als PDF**, oder **beides**. Liegt beides vor, sind die kapitelweisen Markdown-Dateien
führend (sie mappen fast direkt auf die Plattform-Kapitel).

Der Skill **belegt Fachaussagen bevorzugt hieraus** („wo möglich aus dem Skript") und
leitet die **Kapitelstruktur primär aus dem Skript** ab. Nur bei offensichtlichen
Fehlern oder Lücken wird ergänzt (dann `(Ergänzung)`). Die `Vorlesungen/`-Folien sind
dazu **ergänzend**.
