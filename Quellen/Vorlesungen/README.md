# Vorlesungen (primäre Wissensbasis)

Folien/Skript des Moduls als **PDF** oder **PPT/PPTX** ablegen — idealerweise pro
Vorlesung/Kapitel eine Datei, sinnvoll benannt (z. B. `01-Einfuehrung.pdf`).

Der Skill leitet **hieraus** die Kapitelstruktur ab und **belegt Fachaussagen aus
diesem Material**. Nur bei offensichtlichen Fehlern oder fehlenden Infos ergänzt er
allgemeingültiges Wissen (dann als `(Ergänzung)` markiert).
