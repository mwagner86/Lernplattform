# Abgaben (bewertete Zwischenabgaben)

Vom Dozenten **bewertete Zwischen-/Einsendeabgaben** (mit Feedback/Punkten), falls
vorhanden — als PDF, Markdown oder Text.

Der Skill nutzt sie als **Basis**: das **Feedback zeigt Bewertungskriterien und typische
Fehler**. Daraus entstehen bessere Musterlösungen/Worked Examples und „häufige
Fehler"-Hinweise; außerdem schärfen sie die Themenpriorisierung. Gewicht: hoch.
