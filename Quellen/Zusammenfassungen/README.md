# Zusammenfassungen (deine vorhandenen Notizen — optional)

**Bereits vorhandene** eigene Zusammenfassungen/Notizen (Markdown, PDF, Text). Optional.

Wenn hier etwas liegt, nutzt der Skill es als **Zusatzquelle/Startpunkt** für die
generierten Kapitel. Ist der Ordner leer, erzeugt der Skill die Zusammenfassungen
allein aus den `Vorlesungen`.

> Nicht verwechseln mit der **Ausgabe**: Die generierten Plattform-Seiten schreibt der
> Skill in den `contentDir` aus `content.config.json` (standardmäßig `../Inhalte`),
> nicht hierher.
