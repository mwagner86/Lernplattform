# Spickzettel

Dies ist ein **Spickzettel** (Seitentyp `cheatsheet`, referenziert über
`cheatsheetFile` bzw. `cheatsheetFiles` in `content.config.json`). Maximale
Informationsdichte, nur das Prüfungskritische — verdichtet auf möglichst eine Seite.

## Kernbegriffe

- **Begriff A** — Kurzdefinition in einem Halbsatz.
- **Begriff B** — Kurzdefinition. Abgrenzung zu A.
- **Formel** — z. B. `M = E − N + P` (Kürzestform, ohne Herleitung).

## Prinzipien / Listen

1. Prinzip eins (Merksatz)
2. Prinzip zwei
3. Prinzip drei

## Mehrere Spickzettel

Für mehrere Spickzettel (z. B. Kurz + Last-Minute) in `content.config.json` das
Array `cheatsheetFiles` nutzen — sie erscheinen in der angegebenen Reihenfolge
unter „Hilfsmittel".
